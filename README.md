# 🎂 생일 축하 애니메이션

특별한 사람을 위한 도트 스타일 생일 축하 애니메이션 웹 애플리케이션입니다.

포켓몬스터 스타일의 픽셀 아트 애니메이션으로 생일을 축하하는 메시지를 전달할 수 있습니다.

## 📖 프로젝트 소개

이 프로젝트는 생일을 맞은 특별한 사람을 위해 개인화된 생일 축하 애니메이션을 만들어주는 웹 애플리케이션입니다.

### 주요 기능

- 🎨 포켓몬스터 스타일의 도트 애니메이션
- 📱 모바일 최적화
- 🔗 쉬운 공유 기능 (카카오톡, 링크 복사)
- 🎁 개인화된 생일 메시지
- 🎆 아름다운 불꽃놀이 애니메이션

### 사용 방법

1. 이름을 입력하고 공유하기 버튼을 클릭
2. 생성된 링크를 생일인 사람에게 전달
3. 링크를 열면 자동으로 애니메이션이 재생됩니다

### 애니메이션 구성

- **씬 1**: 길을 걷는 장면
- **씬 2**: 언덕 끝에서 도시 바라보기
- **씬 3**: 불꽃놀이만 보이는 하늘
- **씬 4**: 언덕에 앉아서 불꽃놀이 보기
- **최종**: 생일 축하 메시지 표시
