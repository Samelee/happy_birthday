# 기여 가이드라인

생일 축하 애니메이션 프로젝트에 기여해주셔서 감사합니다! 🎉

## 기여 방법

### 1. 이슈 리포트

버그를 발견하셨거나 기능 제안이 있으시면:
- [GitHub Issues](https://github.com/your-username/happy-birthday/issues)에 이슈를 등록해주세요
- 명확한 제목과 설명을 작성해주세요
- 가능하면 스크린샷이나 재현 방법을 포함해주세요

### 2. Pull Request

1. **Fork** 저장소를 포크하세요
2. **Branch** 새 기능 브랜치를 만드세요 (`git checkout -b feature/amazing-feature`)
3. **Commit** 변경사항을 커밋하세요 (`git commit -m 'Add amazing feature'`)
4. **Push** 브랜치에 푸시하세요 (`git push origin feature/amazing-feature`)
5. **Pull Request** Pull Request를 생성하세요

### 3. 코딩 스타일

- JavaScript: ES6+ 문법 사용
- CSS: BEM 네이밍 컨벤션 권장
- HTML: 시맨틱 태그 사용
- 주석: 복잡한 로직에는 주석 추가

### 4. 커밋 메시지

명확하고 간결한 커밋 메시지를 작성해주세요:

```
feat: 새로운 기능 추가
fix: 버그 수정
docs: 문서 수정
style: 코드 포맷팅
refactor: 코드 리팩토링
test: 테스트 추가
chore: 빌드 설정 변경
```

### 5. 테스트

변경사항을 제출하기 전에:
- [ ] 로컬에서 테스트 완료
- [ ] 모바일에서 테스트 완료
- [ ] 다른 브라우저에서 테스트 완료
- [ ] 기존 기능이 정상 작동하는지 확인

## 개발 환경 설정

1. 저장소 클론
2. 로컬 웹 서버 실행
3. 브라우저에서 테스트

## 질문이 있으신가요?

이슈에 질문을 남겨주시거나, 프로젝트 메인테이너에게 직접 연락해주세요.
